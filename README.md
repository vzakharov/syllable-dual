Dual-package port of the `syllable` npm package. TypeScript definitions included.

## Installation

```bash
npm install syllable
```

## Usage

### CommonJS

```js
const { syllable } = require('syllable');

console.log(syllable('Hello World!')); // 3
```

### ES6

```js
import { syllable } from 'syllable';

console.log(syllable('Hello World!')); // 3
```
